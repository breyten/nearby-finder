# Nearby Finder

## Project Documentation

### Features
- Finds nearby locations using various criteria.
- Interactive map interface.
- User-friendly navigation and search options.

### Usage Instructions
1. Clone the repository using `git clone https://github.com/breyten/nearby-finder.git`.
2. Navigate to the project directory: `cd nearby-finder`.
3. Install the necessary dependencies: `npm install`.
4. Start the application: `npm start`.
5. Open your web browser and navigate to `http://localhost:3000`.

### Technical Details
- Built with JavaScript, React, and Node.js.
- Utilizes RESTful APIs for fetching location data.
- Responsive design for mobile and desktop users.
